Projeto Agrinho 2026 - Agro Forte, Futuro Sustentável

Categoria: Programação Front-End (Ensino Médio - 1ª Série)

🎯 Objetivo do Tema

O objetivo deste site é demonstrar como o agronegócio brasileiro utiliza a tecnologia para manter um equilíbrio real entre a produção de alimentos e a preservação do meio ambiente. Através de uma interface interativa, buscamos educar o usuário sobre conceitos como Agricultura 5.0, ILPF e Preservação Ambiental.

🚀 Tecnologias e Funcionalidades

Para atender aos critérios de excelência do concurso, o projeto utiliza:

1.
HTML5 Semântico: Uso correto de tags como <header>, <nav>, <main>, <section> e <footer>.

2.
CSS3 Moderno: Layout responsivo (Flexbox/Grid), variáveis CSS e transições suaves.

3.
JavaScript Funcional:

•
Personalização: Saudação personalizada com o nome do usuário (localStorage).

•
Acessibilidade: Controle de tamanho de fonte e Modo Escuro.

•
IA Hub: Simulação de inteligência artificial com base de conhecimento temática.

•
Quiz Interativo: Sistema de perguntas e respostas para fixação de conteúdo.



📁 Organização do Projeto

•
index.html: Página inicial com visão geral do tema.

•
tecnologia.html: Foco em inovações como drones e sensores.

•
preservacao.html: Detalhes sobre sustentabilidade e leis ambientais.

•
ia.html: Hub interativo com IA e Quiz.

•
/css/style.css: Estilização completa e responsiva.

•
/js/main.js: Lógica do site com comentários didáticos.

•
/img/: Pasta contendo as imagens otimizadas do projeto.

•
Identidade Visual: Design autoral, fugindo de modelos genéricos.

•
Interatividade: O JavaScript manipula o DOM de forma clara e funcional.

•
Responsividade: O site se adapta a celulares, tablets e computadores.

•
Código Limpo: Identação perfeita e comentários explicativos.

•
README Completo: Instruções claras de uso e objetivos.
